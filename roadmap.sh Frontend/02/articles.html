What is HTML?

HTML, short for HyperText Markup Language,
is a markup language used to structure and present
content on the web.

What is CSS?

Cascading Style Sheets (CSS) is a stylesheet
language used to describe the presentation of a
document written in HTML or XML
(including XML dialects such as SVG, MathML
or XHTML).

What is JavaScript?

avaScript is a versatile scripting language primarily
used for creating interactive and dynamic web pages.
It allows developers to add features like animations,
respond to user interactions (like clicks), and
manipulate the content of a webpage without requiring
a full page refresh.
